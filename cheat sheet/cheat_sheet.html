Welcome to the cheat sheet

Tag Name Description
< table > Table The Wrapper element for all HTML tables.
< thead > Table Head The set of rows defining the column headers in a table
< tbody > Table Body The set of rows containing actual table data
< tr > Table Row The table row container
< td > Table Data The table row container
< tfoot > Table Foot The set of rows defing the footer in a table

Basic Tags

Tag Name Description
< html > Html Creates a HTML document
< head > Head Set off the title & other into that isn't displayed
< body > Body Sets off the visible portion of the documnet
< title > Title Puts name of the document in the title bar; when a bookmarking pages, this is what is bookmarked

Formatting

Tag Name Description
< p >< /p > p element Creates a new paragraph
< br > br element AInserts a line break (carriage return)
< blockquote > blockquote Puts content in a quote - indents text from both sides
< div > div Used to format block content with css
< span > Span Used to format inline content with css

Text Tags

Tag Name Description
< pre >< /pre > pre Creates preformatted text
< h1> < /h1 --> < h6 > < /h6 > H1 to H6 Created headlines -- H1=Largest to H6=Smallest
< b > b element Creates bold text (should use < strong > instead)
< i > i element Creates italicized text (should use < em > instead)
< tt > tt element Creates typewriter-style text
< code > code Used to define a source code, usually monospace
< cite > cite Creates a citation, usually pricessed in italics
< address > address Creates address section, usually processed in italics
< em > em element Emphasizes a word (usually processed in italics)
< strong > Strong Emphasizes a word (usually processed in bold)
< font size=? > Font-Size Sets size of front - 1 to 7 (should use CSS instead)
< font color=? > Font-Color Sets font color (should use CSS instead
< font face=? Font-Face Defines the font used (should use CSS instead)
