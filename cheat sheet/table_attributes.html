Table Attributes

Attributes Name Description
colspan Column Span Defines how many columns a td element should span
rowspan Row Span Defines how many rows a td element should span
< table border=?> Table border Set the width of the border around table cells
< table cellspacing=?> Table cellspacing Sets amount of space between table cells
< table cellpadding=?> Table cellpadding Sets amount of space between a cell's border and in contents
< table width=?> Table width Sets width of the table in pixels or as a percentage
< tr align=?> Tr align Sets alignment for cells within in the row (left/center/right)
< td align=?> Td align Sets alignment for cells (left/center/right)
< tr valign = ?> Tr valign Sets vertical alignment for cell (top/middle/bottom
< td rowspan=?> Td rowspan Sets number of rows a cell should span (defualt=1)
< td colspan=? > Td colspan Sets number of columns a cell should span
< nowrap> Nowrap Prevents lines within a cell from being broken to fit
